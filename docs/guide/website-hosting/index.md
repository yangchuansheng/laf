## 简介

静态托管是基于云存储提供的网站托管服务。

可通过「开发控制台」或 `laf-cli` 将网页静态文件上传到云存储的 bucket 中，

静态托管会为你的网站自动提一个独立的域名，也可以绑定自定义域名。


## 如何使用？

- 将开发好的项目打包。
- 创建一个 bucket ，将打包后的项目丢进去。
- 点击开托管，上线成功！
